# Card

完整的校园一卡通系统

## Configure

1 Get the RFID Libarary

``` bash
git submodule init
git submoudle update
```

2 Copy files in `./rfid/src` to the project folder

## 组件说明

| 文件          | 说明     |
| ----------- | ------ |
| CardBackend | 系统后台源码 |
| card        | 读卡器源码  |
| Terminal    | 桌面终端源码 |
| report.md   | 报告     |
| pre.pptx    | 报告ppt  |
